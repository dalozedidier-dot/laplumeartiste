# La Plume Artiste - Site Web

Site personnel de Charlotte Ollivier, journaliste en herbe.

## Installation sur GitHub Pages

1. Créez un nouveau repository sur GitHub
2. Uploadez tous ces fichiers à la racine du repository
3. Allez dans Settings > Pages
4. Sélectionnez la branche "main" et cliquez Save
5. Votre site sera disponible à : `https://votre-nom.github.io/nom-du-repo/`

## Fichiers inclus

- `index.html` - Page d'accueil
- `listing.html` - Liste des articles
- `contact.html` - Page contact
- `article-*.html` - Pages des articles
- `assets/` - CSS, JS, images
